Self Hosted Website

I created a self-hosted website at agnivachaudhuri.com to improve and showcase my skills in Linux system administration, devops, and web development. The website also serves as a staging ground for any prototypes/concepts that involve internet services.

Process/Design

I used DigitalOcean cloud services to host the website, then nginx as the actual web server. I did the following:

  • Created an Ubuntu Server virtual machine using DigitalOcean services
  • Set up nginx and configure it to handle web requests
  • Purchase domain from Cloudflare and point it at appropriate IP address

The project is still in progress and I have not set up everything yet. I will do the following in the near future:

  • Mirror this website (agnivachaudhuri.github.io) onto agnivachaudhuri.com
  • Set up messaging server
  • Create an account system so that authorized users can access protected data
